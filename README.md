Aura-Link-A-Full-Stack-Autonomous-UAV-with-Custom-Silicon-Gesture-AI

description

Aura-Link is a high-performance quadcopter engineered from the component level to explore the intersection of power electronics and computer vision. Moving beyond "off-the-shelf" components, this project involves designing and soldering custom Electronic Speed Controllers (ESCs) from scratch using high-current MOSFET power stages and a Raspberry Pi-based Flight Controller running a custom PID stabilization loop.

The drone features three distinct flight profiles: a raw Manual Mode for high-maneuverability, a Semi-Automatic Mode utilizing input-smoothing algorithms for cinematic stability, and a fully Autonomous AI Mode. Driven by a Raspberry Pi and a Pi Camera, the drone utilizes MediaPipe/OpenCV to recognize human hand gestures, allowing for "Follow Me" tracking and autonomous orbiting for 360-degree aerial photography. To complete the ecosystem, the project includes a custom-built 2.4GHz remote transmitter designed to interface directly with the proprietary flight software.
